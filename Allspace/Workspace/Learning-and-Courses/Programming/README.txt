также
В этой папке находятся папки с изучениям какой-то технологии по понкретному курсу. 

Сначала - технология, потом - курс, далее - его автор (по значимости, это относится к методологии давать названия папкам / проектам).
* Например, папки 
 - vue
 -- learn-vue-vladilen
 -- learn-vue-tronsy
 - html/css
 -- learn-html-css-js-fls

* В этой папке не находятся туториалы по программам, например IDEA.
