В этой паке находятся папки с изучениям какой-то технологии по покнретному курсу. 

Сначала технология потом - курс, его автор (по значимости, это относится к методологии давать названия папкам / проектам).
* Например, папки 
 - vue
 -- learn-vue-vladilen
 -- learn-vue-tronsy
 - html/css
 -- learn-html-css-js-fls

* В этой папке не неаходятся туториалы по программам, например IDEA.
